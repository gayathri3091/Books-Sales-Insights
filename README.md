# Project Title:
Book Sales Trend And Revenue Insights
## Description:
This project is to analyse book sales trend by genre/author, also to visualize the sales and revenue insights using KPIs,map,bar/line charts.The aim of this project is to improve sales and revenue overall by period of time.
### Tools used:
-MySQL Workbench is used to find insights through query.
-PowerBI is used to visualize the insights through column chart,line chart,MAP chart and KPIs with DAX formulas. 
#### Visualization Insights:
-KPIs card visuals are used to show total Stock,total Revenue,total sales.
-Slicers are used to choose year according insights. 
-MAP chart is used to display customers by country/city.
-Bar/line chart are used to show book sales by genre & YTD,QTD respectively.
